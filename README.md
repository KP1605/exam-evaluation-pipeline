# AI-Based Exam Evaluation Input Layer

## Overview
This project implements the input layer of an AI-powered exam evaluation pipeline.

## Features
- Admin upload (question paper + answer key)
- Single & bulk student upload
- OCR simulation & data extraction
- Verification workflow
- Responsive UI

## Architecture
- Presentation Layer (React components)
- Management Layer (state via reducer)
- Service Layer (API & extraction logic)

## Tech Stack
- React (Vite)
- Tailwind CSS
